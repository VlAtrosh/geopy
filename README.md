# Geopy (mini-project)

1. Краткое описание инструмента  
`geopy` — это библиотека для Python, которая помогает работать с геокодированием и обратным геокодированием через разные сервисы, а ещё включает готовые функции для вычисления расстояний по поверхности Земли.

2. Цель изучения  
Освоить инструмент не “в лоб”, а как часть инженерного пайплайна: научиться превращать адреса в координаты, контролировать качество результата и понимать ограничения провайдеров.

3. Описание реализованной задачи  
Скрипт читает список адресов из `input.csv`, геокодирует каждый адрес (через Nominatim), считает расстояние до заданной точки интереса и сохраняет результат в `output.csv`. Для повторных запусков используется простой кэш, чтобы не делать одинаковые запросы снова.

4. Инструкция по запуску  
Установи зависимости: `py -m pip install -r requirements.txt`  
Запусти: `py main.py --input input.csv --output output.csv`  
Если нужно, можно поменять точку интереса:  
`py main.py --poi-lat 55.7558 --poi-lon 37.6173`
При обращениях к Nominatim рекомендуется передавать `--user-agent` с контактными данными.

5. Общие выводы  
geopy удобен для учебных и небольших инженерных задач, где нужно быстро “подключить географию” к данным. При этом важно помнить про лимиты сервиса, качество входных адресов и обработку случаев, когда провайдер не находит нужную локацию.
