# Использованные источники

- Официальная документация Geopy: https://geopy.readthedocs.io/en/stable/ — базовое понимание интерфейсов геокодеров, объектов результата и модулей расстояний.
- Раздел Geocoders в документации Geopy: https://geopy.readthedocs.io/en/stable/#module-geopy.geocoders — примеры прямого и обратного геокодирования, поля возвращаемых объектов.
- Раздел Distance в документации Geopy: https://geopy.readthedocs.io/en/stable/#module-geopy.distance — формирование вычислений расстояний по поверхности Земли.
- Rate Limiter в документации Geopy: https://geopy.readthedocs.io/en/stable/#module-geopy.extra.rate_limiter — подход к ограничению частоты запросов и уменьшению количества сбоев.
- Документация Nominatim (Usage Policy): https://operations.osmfoundation.org/policies/nominatim/ — правила использования сервиса и причины, почему запросы нужно ограничивать.
- Документация Nominatim API: https://nominatim.org/release-docs/latest/api/ — детали параметров поиска и формат ответа геокодера.
